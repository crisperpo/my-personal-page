- React.js
- TypeScript

### Backend
- Node.js
- Express

### Infrastructure
- Vercel (hosting)
- GitHub Actions (CI/CD)

### Development Tools
- ESLint + Prettier
- Vitest + React Testing Library
- Vite
- pnpm
